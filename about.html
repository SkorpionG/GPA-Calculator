About GPA Calculator

A simple, fast, and bilingual GPA calculation tool built with modern, standards-based web technologies — no frameworks, no dependencies, just the web platform.

Project Overview

GPA Calculator is a client-side web application that helps students calculate their semester GPA using a standard 4.3-point scale. All course data is stored locally in your browser via localStorage, ensuring your privacy with no server calls or accounts required.

Features

  • Real-time GPA calculation
  • 4.3-point GPA scale (A+ to F)
  • Sort subjects by grade
  • Persistent local storage
  • Bilingual: English & 繁體中文
  • Animated intro with toggle
  • Shared header via Web Components
  • Responsive design
